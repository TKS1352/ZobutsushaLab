# 造物者ラボ ランディングページ

古と今を結ぶ、独創のソリューションを提供する合同会社 造物者ラボのランディングページです。

## 🎨 デザインコンセプト

- **古代と現代の融合**: 伝統的な美意識と最先端のテクノロジーを融合
- **カラーパレット**: 温かみのあるアースカラーと金属的なアクセント
- **タイポグラフィ**: 優雅なセリフ体とモダンなサンセリフ体の組み合わせ
- **レスポンシブデザイン**: モバイルファーストでタッチフレンドリー
- **アクセシビリティ**: WCAG 2.1 AA準拠のコントラスト比とキーボードナビゲーション

## 🚀 技術スタック

- **Vite**: 高速な開発サーバーとビルドツール
- **Vanilla HTML/CSS/JS**: フレームワークに依存しない軽量な実装
- **CSS Variables**: 一貫したデザインシステム
- **Intersection Observer**: パフォーマンスに優れたスクロールアニメーション

## 📂 プロジェクト構造

```
landing-page/
├── index.html          # メインHTMLファイル
├── src/
│   ├── style.css      # スタイルシート
│   └── main.js        # JavaScriptロジック
├── package.json       # 依存関係の定義
└── README.md          # このファイル
```

## 🛠️ 開発方法

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてプレビューできます。

### 本番ビルド

```bash
npm run build
```

ビルド成果物は `dist/` ディレクトリに生成されます。

### ビルドのプレビュー

```bash
npm run preview
```

## 📄 ページセクション

1. **Hero**: 会社名とキャッチコピー、CTAボタン
2. **About**: 企業理念とビジョン、古代と現代の融合というストーリー
3. **Services**: システム部門と芸術部門の事業内容
4. **Strengths**: 独創性と迅速な対応を示す3つの強み
5. **Company**: 会社概要（基本情報）
6. **Contact**: お問い合わせフォームとメールアドレス
